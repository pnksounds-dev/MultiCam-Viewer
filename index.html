Direct USB camera — no app on your phone

  1. Connect your phone with a data USB cable
  2. Enable Developer Options: Settings → About phone → tap "Build number" 7 times
  3. Enable Settings → System → Developer Options → USB Debugging
  4. On the phone popup "Allow USB debugging?" → tap Allow (tick "always allow")
  5. Click ↻ Refresh above — your phone appears as "📱 [model] — back camera"
  6. Select it. The first time, watch your phone for a camera permission prompt → Allow

This uses scrcpy (bundled). A temporary helper is pushed to the phone over USB and removed on disconnect — nothing is permanently installed.

Built-in USB Webcam mode (Android 14–16)

  1. Plug phone into PC with a data USB cable
  2. On phone: pull down the notification shade → tap the USB notification
  3. Choose "USB Webcam" (under "Use USB for")
  4. Click ↻ Refresh — phone appears as "🎥 ... Webcam"

⚠ If "USB Webcam" isn't listed, your device doesn't expose it — use the USB Debugging tab instead (works on all Android 12+).

Virtual Camera Driver (UnityCapture)

To output video as a virtual webcam that OBS can detect, the UnityCapture DirectShow filter must be registered on your PC.

DLL must be in the vcam/ folder — accepted filenames:

  • UnityCaptureFilter64.dll  ✓ found or ✗ missing

Clicking Register shows a Windows UAC admin prompt — click Yes to allow.

Virtual Camera Slots

Each open window outputs to a different slot in OBS:

  • Window 1 → MultiCam (or "Unity Video Capture" if using the original DLL)
  • Window 2 → MultiCam 2
  • Window 3 → MultiCam 3

Recommended: OBS Window Capture is more reliable because it does not need the native frame bridge.

Using with OBS

  1. Recommended — no driver needed: Add Source → Window Capture → select "MultiCam Viewer"
  2. Alternative — register the driver above and approve the UAC prompt
  3. In this app: select your phone camera in the dropdown
  4. In OBS: Add Source → Video Capture Device
  5. Device: "MultiCam" (or "Unity Video Capture" if using the original DLL)
  6. Second phone: click + New Camera, add another OBS source → "MultiCam 2"

Window Capture (no driver needed)

Select your phone in the dropdown, then in OBS: Add Source → Window Capture → select "MultiCam Viewer". Open a second app window for the second phone.
