Direct USB camera — no app on your phone

  1. Connect your phone with a data USB cable
  2. Enable Developer Options: Settings → About phone → tap "Build number" 7 times
  3. Enable Settings → System → Developer Options → USB Debugging
  4. On the phone popup "Allow USB debugging?" → tap Allow (tick "always allow")
  5. Click ↻ Refresh above — your phone appears as "📱 [model] — back camera"
  6. Select it. The first time, watch your phone for a camera permission prompt → Allow

This uses scrcpy (bundled). A temporary helper is pushed to the phone over USB and removed on disconnect — nothing is permanently installed.

Built-in USB Webcam mode (Android 14–16)

  1. Plug phone into PC with a data USB cable
  2. On phone: pull down the notification shade → tap the USB notification
  3. Choose "USB Webcam" (under "Use USB for")
  4. Click ↻ Refresh — phone appears as "🎥 ... Webcam"

⚠ If "USB Webcam" isn't listed, your device doesn't expose it — use the USB Debugging tab instead (works on all Android 12+).

Phone Capture ?

Uses USB debugging (ADB) + bundled scrcpy to pull the camera feed directly. Requires Android 12+, USB Debugging enabled, and tapping Allow on the phone prompts.

OBS Setup ?

  1. Select your phone camera in the dropdown above
  2. In OBS: Add Source → Window Capture → select "MultiCam Viewer" (recommended, no driver needed)
  3. Or register the driver above, then OBS → Add Source → Video Capture Device → device "MultiCam"
  4. Second phone: click + New Camera, add another OBS source → "MultiCam 2"
